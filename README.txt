The Connect Four game was built in C# .NET using Visual Studio 2015 and Resharper. It is a console application.

The solution file is in the ConnectFour folder.

The Unit and Component tests are dependent on NUnit 3.4.1 and Moq 4.5.10, Moq is dependent on Castle.Core 3.3.3.

There is a stand alone executable file that can be used to run the game.

It was built using TDD and following SOLID design principles.

A stand alone executable file can be found in Release folder. Simply double click the "ConnectFour.App.exe" file to run the program.

Upon victory or draw the game will close after the next user input.
